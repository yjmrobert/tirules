## Rules Reference

  
A structure is a type of unit. PDS units and space docks are structures.

 
 1. Structures are always placed on planets.
 
    1. The Clan of Saar’s “Floating Factory” faction–specific space dock is placed in a system’s space area.

 2. Structures are primarily placed on planets using the <i>Construction</i> strategy card.
 3. Structures cannot move or be transported.
 4. A player can have a maximum of <b>one</b> space dock on each planet.
 5. A player can have a maximum of <b>two</b> PDS units on each planet.
 6. A player cannot place a structure on a planet if it would exceed the maximum number of allowed structures of that type on that planet.
